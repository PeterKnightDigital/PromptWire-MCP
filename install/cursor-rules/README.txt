PromptWire — Cursor project rules

Copy promptwire.mdc into your ProcessWire project's .cursor/rules/ folder:

  cp install/cursor-rules/promptwire.mdc /path/to/pw-site/.cursor/rules/

The rule tells the AI agent to use PromptWire MCP for CMS content and to edit
source files only for templates, modules, and styling.

For global coverage across all ProcessWire projects without copying this file,
use the personal skill in the PromptWire docs (README § Cursor rules).
